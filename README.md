## Landscape of colonization factors in human gut microbiome and their association with inflammatory bowel disease

- pipeline: metagenomic (MGX) abundance and metatranscriptomic (MTX) expression estimation, as well as CF gene identification in UHGP-90 and UHGG catalogs
- scripts: some R function used in this program
- data: the intermediate data during analysis
- FigureX: the scripts and necessary files for each figure
